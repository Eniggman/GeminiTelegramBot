# 🤖 Google Gemini Telegram Bot

![Python](https://img.shields.io/badge/Python-3.10%2B-3776AB?style=for-the-badge&logo=python&logoColor=white)
![Gemini](https://img.shields.io/badge/Google%20AI-Gemini-8E75B2?style=for-the-badge&logo=google&logoColor=white)
![Telegram](https://img.shields.io/badge/Telegram-Bot-2CA5E0?style=for-the-badge&logo=telegram&logoColor=white)

Telegram бот для общения с Gemini, написанный на Python с использованием Antigravity IDE (Google).
Поддерживает мультимодальное общение (текст, фото, аудио), генерацию изображений и поиск в интернете. От вас требуется API Gemini, и сервер на котором он будет работать

---

## ✨ Ключевые Возможности

| Иконка | Функция | Описание |
| :---: | :--- | :--- |
| 💬 | **Умный диалог** | Поддержка контекста беседы и разных личностей. |
| 🖼️ | **Видение (Vision)** | Понимает и описывает отправленные фотографии. |
| 🎨 | **Генерация** | Создает изображения по текстовому описанию (Imagen 3). |
| 🖌️ | **Редактирование** | Изменяет фото по инструкции пользователя. |
| 🌐 | **Grounding** | Ищет актуальную информацию в Google Поиске. |
| 📄 | **Документы** | Анализирует PDF, TXT, CSV, JSON файлы. |
| 🎙️ | **Голос** | Распознает голосовые сообщения и отвечает на них. |

## 🕹 Управление и Команды

### 🔥 Специальные Префиксы
*Пишутся в начале сообщения для быстрых действий.*

| Префикс | Аргумент | Описание | Пример |
| :--- | :--- | :--- | :--- |
| **К** / **Картинка** | `<промпт>` | Генерация изображения по описанию. | `К рыжий кот в космосе` |
| **Пр** / **Перевод** | `<текст>` | Быстрый перевод на русский язык. | `Пр How are you?` |
| **Р** / **Редактировать** | `<инструкция>` | Редактирование прикрепленного фото. | *(с фото)* `Р добавь очки` |
| **.** | **/start** | Сброс контекста диалога (забыть старое). | `. ` |

### 📸 Работа с Фото

**При отправке фото без подписи** появляются инлайн-кнопки (⏱ таймаут 2 минуты):

| Кнопка | Модель | Функция |
| :--- | :--- | :--- |
| 🔍 **Анализировать** | Выбранная модель (💎 Pro / ⚡ Flash) | Описывает содержимое изображения |
| ✏️ **Редактировать** | `gemini-3-pro-image-preview` (всегда Pro) | Редактирует фото по вашему запросу |

**Как это работает:**
1. Отправьте фото → появятся кнопки
2. Нажмите **Анализировать** → бот опишет изображение
3. **Контекст сохранён!** Теперь можно задавать вопросы об этом изображении текстом в течение 5 минут
4. Нажмите **Редактировать** → введите описание изменений → получите новое фото

**🧠 Контекст изображения:**
> После анализа изображения (кнопка 🔍) бот **сохраняет картинку в памяти на 5 минут**.  
> В течение этого времени все текстовые вопросы будут обрабатываться с учётом изображения.  
> **Пример:** Отправили фото игры → нажали "Анализировать" → спросили "что это тропи?" → бот ответит с учётом картинки! ✨

**⏱ Автоматический сброс контекста:**
- ✅ Через **5 минут** после анализа — автоматически
- ✅ Команда `.` — мгновенно сбрасывает весь контекст (текст + изображение)
- ✅ Команда `/start` — полный сброс сессии

> **⚠️ Таймаут кнопок:** Если не нажать кнопку в течение 2 минут, данные фото удаляются автоматически.

### 🛠 Основные Команды

*   `/start` — 👋 **Начать** новый диалог и сбросить память.
*   `/help` — ℹ️ **Справка** по всем командам.
*   `/status` — 📊 **Статистика** бота и состояние.
*   `/models` — 🧠 **Список** всех доступных AI моделей.
*   `/id` — 🆔 **Узнать** свой Telegram ID.

### ⚙️ Настройка Моделей

*   `/1model` — Включить **Gemini Pro** (💎 Умнее, но медленнее).
*   `/2model` — Включить **Gemini Flash** (⚡ Быстрее и экономнее).
*   `/imagepro` — Включить **Pro** для картинок.
*   `/imageflash` — Включить **Flash** для картинок.

## 🧠 Используемые Модели

| Категория | 💎 Pro Версия | ⚡ Flash Версия |
| :--- | :--- | :--- |
| **Текстовое ядро** | `gemini-3-pro-preview` | `gemini-3-flash-preview` |
| **Генерация фото** | `gemini-3-pro-image-preview` | `gemini-2.5-flash-image` |

> **ℹ️ Примечание:** Редактирование фото (кнопка "✏️ Редактировать") всегда использует `gemini-3-pro-image-preview` независимо от настроек `/imagepro` или `/imageflash`.
