▶ СПЕЦИАЛЬНЫЕ ПРЕФИКСЫ (пишутся в начале сообщения)
───────────────────────────────────────────────────────────────

К <описание>     → Генерация изображения
                   Способ 1: Отправь "К", потом описание
                   Способ 2: Сразу "К красивый закат"

Пр <текст>       → Перевод текста (одноразовый)
Перевод <текст>  → То же самое
                  

Р <инструкция>   → Редактирование фото (прикрепить фото!)
                   Пример: (отправить фото) Р сделай чёрно-белым

С                → Анализ фото 
Смотри           → То же самое
                   Способ 1: Отправь "С", потом фото
                   Способ 2: Отправь фото с подписью "С"

.                → Сброс контекста диалога
                   (просто отправь точку)


▶ КОМАНДЫ ЧЕРЕЗ /
───────────────────────────────────────────────────────────────

/start           → Сброс контекста + приветствие
/help            → Справка по командам
/id              → Показать свой Telegram ID
/status          → Статус бота и статистика

/1model          → Переключить на Pro модель (текст)
/2model          → Переключить на Flash модель (текст)

/imagepro        → Pro модель для генерации картинок
/imageflash      → Flash модель для генерации картинок

/models          → Показать все доступные модели Gemini


▶ АДМИН КОМАНДЫ (только для ADMIN_ID)
───────────────────────────────────────────────────────────────

/add <id>        → Добавить пользователя
                   Пример: /add 123456789

/del <id>        → Удалить пользователя
                   Пример: /del 123456789


▶ ГОЛОСОВЫЕ СООБЩЕНИЯ
───────────────────────────────────────────────────────────────

Просто отправь голосовое → Бот распознает и ответит


МОДЕЛИ:

Текст:
  Pro   = gemini-3-pro-preview (умный, медленнее)
  Flash = gemini-flash-latest (быстрый)

Картинки:
  Pro   = gemini-3-pro-image-preview (качество)
  Flash = gemini-2.5-flash-image (скорость)
