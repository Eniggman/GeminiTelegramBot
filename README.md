# 🤖 Google Gemini Telegram Bot

![Python](https://img.shields.io/badge/Python-3.10%2B-3776AB?style=for-the-badge&logo=python&logoColor=white)
![Gemini](https://img.shields.io/badge/Google%20AI-Gemini-8E75B2?style=for-the-badge&logo=google&logoColor=white)
![Telegram](https://img.shields.io/badge/Telegram-Bot-2CA5E0?style=for-the-badge&logo=telegram&logoColor=white)

Функциональный Telegram бот, написанный на Python с использованием Antigravity IDE (Google).  
Поддерживает мультимодальное общение (текст, фото, аудио), генерацию изображений и поиск в интернете.

---

## ✨ Ключевые Возможности

| Иконка | Функция | Описание |
| :---: | :--- | :--- |
| 💬 | **Умный диалог** | Поддержка контекста беседы и разных личностей. |
| 🖼️ | **Видение (Vision)** | Понимает и описывает отправленные фотографии. |
| 🎨 | **Генерация** | Создает изображения по текстовому описанию (Imagen 3). |
| 🖌️ | **Редактирование** | Изменяет фото по инструкции пользователя. |
| 🌐 | **Grounding** | Ищет актуальную информацию в Google Поиске. |
| 📄 | **Документы** | Анализирует PDF, TXT, CSV, JSON файлы. |
| 🎙️ | **Голос** | Распознает голосовые сообщения и отвечает на них. |

## 🕹 Управление и Команды

### 🔥 Специальные Префиксы
*Пишутся в начале сообщения для быстрых действий.*

| Префикс | Аргумент | Описание | Пример |
| :--- | :--- | :--- | :--- |
| **К** | `<промпт>` | **Картинка**: Генерирует изображение. | `К рыжий кот в космосе` |
| **Пр** | `<текст>` | **Перевод**: Быстрый перевод на русский. | `Пр How are you?` |
| **Р** | `<инструкция>` | **Редакт**: Изменяет прикрепленное фото. | *(с фото)* `Р добавь очки` |
| **.** | *(нет)* | **Сброс**: Забыть контекст диалога. | `.` |

### 🛠 Основные Команды

*   `/start` — 👋 **Начать** новый диалог и сбросить память.
*   `/help` — ℹ️ **Справка** по всем командам.
*   `/status` — 📊 **Статистика** бота и состояние.
*   `/models` — 🧠 **Список** всех доступных AI моделей.
*   `/id` — 🆔 **Узнать** свой Telegram ID.

### ⚙️ Настройка Моделей

*   `/1model` — Включить **Gemini Pro** (💎 Умнее, но медленнее).
*   `/2model` — Включить **Gemini Flash** (⚡ Быстрее и экономнее).
*   `/imagepro` — Включить **Pro** для картинок.
*   `/imageflash` — Включить **Flash** для картинок.
